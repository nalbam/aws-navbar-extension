Identify AWS Regions
At a Glance

Stop deploying to the wrong region. AWS Colorful Navbar adds visual indicators to your AWS Console with colorful gradients and country flags.

Install for Chrome Install for Firefox

Features

🎨

Dynamic Gradients

Each region displays a unique color gradient inspired by its country's flag. Instantly recognize your current region.

🏴

Country Flags

Flags appear next to the region selector. 29 countries and 38+ AWS regions supported.

🎨

Custom Colors

Customize gradient colors with the built-in color picker. Choose 2-4 colors per region.

⚡

Real-time Updates

Changes apply instantly. No page refresh needed.

🌓

Dark/Light Theme

Popup interface supports both dark and light themes.

🔒

Privacy Focused

No data collection. No external requests. Completely local.
